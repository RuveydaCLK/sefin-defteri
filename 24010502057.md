## 📱 Uygulama Hakkında

*Şefin Defteri*, geleneksel yemek kültürünü dijital dünyaya taşıyan bir tarif platformudur. Şefler kendi tariflerini paylaşabilir, kullanıcılar kategorilere göre tarif keşfedebilir, beğendiklerini kaydedebilir.

### ✨ Özellikler

- 🏠 *Ana Sayfa* — Günün tarifi ve son eklenen tarifler
- 📖 *Tarifler* — Kategori ve filtrelerle tarif keşfi
- ➕ *Tarif Ekle* — Şefin kendi tarifini yayınlaması
- ❤️ *Kaydedilenler* — Favori tariflere hızlı erişim
- 👨‍🍳 *Profil* — Şef profili ve istatistikler
- 📋 *Tarif Detay* — Malzemeler ve adım adım yapılış

---


## 🛠️ Kullanılan Teknolojiler ve Kütüphaneler

Projenin performanslı, kararlı ve görsel olarak üst seviyede olması için aşağıdaki popüler kütüphaneler ve araçlar kullanılmıştır:

- *React 19* + *TypeScript*
- *Vite* — Build aracı
- *Tailwind CSS* — Stil
- *Framer Motion* — Animasyonlar
- *Lucide React* — İkonlar
- *GitHub Pages* — Hosting

### 1. Çekirdek Teknolojiler
*   **React 19**: Modern bileşen tabanlı kullanıcı arayüzü kütüphanesi. En son performans iyileştirmelerini ve hook mimarisini barındırır.
*   **TypeScript**: Kod güvenliğini artırmak, tip hatalarını derleme aşamasında yakalamak ve kusursuz bir geliştirme deneyimi sunmak için kullanılmıştır.
*   **Vite**: Son derece hızlı çalışan yeni nesil ön-yüz derleyicisi ve geliştirme sunucusu.

### 2. Tasarım ve Animasyon (Tasarım Sistemi)
*   **Tailwind CSS v4**: Utility-first CSS kütüphanesi. Uygulamanın tüm modern yerleşimi, renk paletleri (soft krem ve sıcak turuncu detaylar), gölgelendirmeleri ve responsive (mobil uyumlu) yapısı saf Tailwind sınıfları ile tasarlanmıştır.
*   **Motion (Framer Motion) (`motion/react`)**: Ekran geçişleri, bento-grid açılışları, butonların basılma hissi ve modal/detay pencerelerinin pürüzsüz animasyonlarla açılmasını sağlayan profesyonel animasyon kütüphanesi.
*   **Lucide React (`lucide-react`)**: Uygulama içindeki tüm modern, minimalist ve yüksek çözünürlüklü vektörel SVG ikon paketini sunar (Ev, Kalp, Kitap, Artı, Arama vb.).

## 📂 Proje Dosya Yapısı 

Şefin Defteri uygulaması, modern bileşen tabanlı tasarımı, güçlü durum yönetimi ve Tailwind CSS entegrasyonu ile modüler bir yapıda geliştirilmiştir. Projenin genel klasör ve dosya hiyerarşisi şu şekildedir:

```text
sefin-defteri/
├── 📁 assets/                  # Uygulamada kullanılan statik görsel ve logolar
├── 📁 src/                     # Uygulamanın tüm kaynak kodları
│   ├── 📁 components/          # Bağımsız ekranlar ve arayüz bileşenleri
│   │   ├── 📄 AuthScreen.tsx          # Şık giriş ve kayıt olma ekranı
│   │   ├── 📄 HomeScreen.tsx          # Günün tarifini (Lazanya) ve öne çıkanları gösteren ana sayfa
│   │   ├── 📄 RecipesScreen.tsx       # Detaylı arama ve kategori filtrelemeli tarif listesi
│   │   ├── 📄 RecipeDetailScreen.tsx  # Porsiyon hesabı ve yapılış adımlı detay ekranı
│   │   ├── 📄 AddRecipeScreen.tsx     # Kullanıcıların yeni tarif ekleyebileceği form
│   │   ├── 📄 SavedScreen.tsx         # Beğenilen/Kaydedilen tarifler sekmesi
│   │   ├── 📄 ProfileScreen.tsx       # Kullanıcı istatistikleri, popüler paylaşımlar ve çıkış (logout) butonu
│   │   └── 📄 LucideIcon.tsx          # Dinamik ikon render desteği sağlayan yardımcı bileşen
│   │
│   ├── 📄 App.tsx              # Ana yönlendirici (Router), alt menü (bottom navbar), üst bar ve global state yönetimi
│   ├── 📄 data.ts              # Ana tarif havuzu, kategoriler ve başlangıç kullanıcı verileri
│   ├── 📄 types.ts             # Proje genelinde kullanılan TypeScript arayüz tanımları (Recipe, User vb.)
│   ├── 📄 index.css            # Küresel CSS kuralları, yazı tipi tanımları ve Tailwind CSS içe aktarımları
│   └── 📄 main.tsx             # React uygulamasının başlangıç noktası (Render root)
│
├── 📄 index.html               # Uygulamanın ana HTML şablonu
├── 📄 vite.config.ts           # Hızlı derleme için Vite yapılandırma dosyası
├── 📄 tsconfig.json            # TypeScript derleyici kuralları ve optimizasyonları
├── 📄 package.json             # Bağımlılıklar (React, Motion, Lucide) ve terminal komutları
├── 📄 .env.example             # Çevre değişkenleri örnek şablonu
└── 📄 .gitignore               # Sürüm kontrolüne (Git) dahil edilmeyecek dosyaların listesi
```

## 🚀 Lokal Kurulum ve Çalıştırma Adımları

Projeyi yerel bilgisayarınızda çalıştırmak veya derlemek için aşağıdaki adımları takip edebilirsiniz:

### 1. Gereksinimler
Bilgisayarınızda **Node.js** (v18 veya üzeri önerilir) ve **npm** paket yöneticisinin kurulu olduğundan emin olun.

### 2. Bağımlılıkların Kurulması
Proje klasörünün içerisine girin ve terminalinizde aşağıdaki komutu çalıştırarak tüm kütüphaneleri yükleyin:
```bash
npm install
```

### 3. Geliştirme (Development) Sunucusunu Başlatma
Uygulamayı yerel olarak canlı önizleme modunda çalıştırmak için:
```bash
npm run dev
```
Bu komut, yerel geliştirme sunucusunu başlatır. Terminalde belirtilen adresi (genellikle `http://localhost:3000` veya `http://localhost:5173`) tarayıcınızda açarak uygulamayı test edebilirsiniz.

### 4. Üretim Sürümü İçin Derleme (Production Build)
Uygulamayı yayına almak amacıyla optimize edilmiş statik dosyaları oluşturmak için:
```bash
npm run build
```
Derleme işlemi bittikten sonra, oluşturulan dosyaları yerel olarak test etmek için aşağıdaki komutla üretim sunucusunu başlatabilirsiniz:
```bash
npm run start
```
